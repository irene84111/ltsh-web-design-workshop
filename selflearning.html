網頁設計自學之路

Web Design Self-learning

成為網頁設計師是門漫長的路。其實技術永遠是無遠弗屆的，唯一能限制自己的其實是你的想像力與創意。換句話說，「程式技術」是必備的基礎，但要脫穎而出，「設計思考」、「想像力」與「創意」才是決勝的關鍵喔！

入門教學網站


  1. Stack Overflow：世界上最大的程式語言交流平台，遇到程式問題可以在這裡找解答。
  2. Codecademy：非常推薦的程式語言自學平台，滿分推薦！
  3. Code School：一樣是很精彩的程式語言自學平台，品質更高，滿滿分推薦！
  4. W3schools：網頁程式語言的小百科全書（精簡版）
  5. 去書局挑本順眼的HTML, CSS or Javscript教學書，也是個好方法，不過書籍的內容有限，網路資源無限\O/

進階學習資源

  1. MDN：網頁程式語言的大英百科全書（專家版）
  2. CSS Tricks：進階CSS技術
  3. Learn Layout：CSS排版教學
  4. HTML5Rocks：進階HTML教學

網頁設計觀摩

  1. Awwwards：網頁設計大獎項
  2. CSS Design Awards：重視CSS表現技法的獎項
  3. Codepen：小而神的作品集散地

學完HTML 和 CSS之後呢？

  1. 學學Javascript。
    他是非常無敵重要的網頁技術，處理更進階更深厚的網頁互動功能，上面分享的網站都有教 Javascript 喔！
  2. 學一些好用的框架(Framework)。
    妥當的使用框架可以幫助你快速打造優質的網站，前提是你要先搞懂那個框架的使用方法與核心哲學。
    1. CSS框架：960 Grid System , Skeleton , Pure.css
    2. Javascript框架：jQuery , React.js
    3. 多功能框架：Bootstrap
  3. 練習規劃完整的網站。
  4. 多看看得獎的作品。
  5. 不斷吸收一些平面設計、互動設計的知識、書籍、作品與思維。
  6. 網頁設計鐵三角都熟了之後，可以試著學一些「後端」技術。

有哪些 HTML & CSS 框架值得一學？

  1. 960 Grid System：經典網格排版框架。
  2. Bootstrap：超夯響應式全套框架。
  3. Skeleton：羽量級響應式微框架。

Javascript 可以學什麼？

  1. 建議先去書局挑本講 Javascript 的書來看，練習寫些書中教的範例，語法熟悉後，再學一些相關框架或技術。
  2. jQuery：非常方便的JS函式庫，幫助你更簡單的做出互動與元素操縱。
  3. Paper.js：功能很完整的互動技術。
  4. D3.js：資料視覺化技術
  5. Three.js：網頁動態3D技術
  6. React.js：Facebook開發的前端框架

有什麼後端技術或程式語言值得一學？

  1. Ruby on Rails：大概是當今最夯的後端框架吧！好用！使用Ruby語言
  2. Django：使用Python程式語言的框架
  3. Laravel：使用PHP語言的框架

上述的四種程式語言：JavaScript、Ruby、Python、PHP都是架網站常用到的語言，其中Javascript是一定要會的，因為他是唯一能在瀏覽器端執行的程式語言。剩下三者是在伺服器端執行，可以挑一個來學即可，這樣就可以做很完整的網站囉。
