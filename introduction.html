網頁設計是什麼

What is Web Design

網頁設計（Web Design），廣義來說就是從「網站規劃」到「網站上線」的一套流程，即網站的「前、後端」，狹義來講則是單指設計網頁的視覺、外觀、樣式、互動效果等等，及所謂網站「前端」的部分。

一般要做一個完整的網站，會把工作內容分為「前端」與「後端」兩部分，前端負責使用者看得到到東西（如圖文內容、按鈕、外觀、色彩、互動效果等等），後端負責處理使用者看不到的程式邏輯（如資料庫操作、會員系統內部程式、線上付款流程等等）。

我們這章主要討論「前端」的部分，也就是網頁設計的狹義定義。

網頁運作的「原理」

每個網站背後其實都是由很多文字檔（裡面寫著程式碼）所組成的，他們放在世界上某個角落的某台伺服器裡。你在瀏覽器打的網址，其實就是連到某台伺服，然後讀取裡面的「某個檔案」（例如index.html），再把那份檔案裡的程式碼傳到你的瀏覽器，在這裡進行解析（類似翻譯的感覺），然後就變成你在瀏覽器看到、摸得到的東東囉！

網頁設計的「語言」

網頁設計主要由三種語言所構成，HTML、CSS與JavaScript，這三者分別負責網頁設計的不同通能，以下一一介紹。

HTML

HTML 是一種「標示語言」（ markup language ），用來建構網頁的主架構、圖文內容，是一個網頁的主體。副檔名為.html，一般瀏覽器打開的網頁就是開啟某個.html檔


CSS

HTML建構的網頁是沒有任何樣式的，也就是完全的白底黑字、沒有其他顏色、沒有排版...就只是一個只有圖文內容的文件，但這種網站沒有人要看，所以出現了CSS。CSS是一種樣式語言，拿來描述HTML元素的外觀樣式，讓醜醜單調的網頁增添更多色彩與視覺設計元素。

CSS語法可以內嶔在.html裡，也可以放在.html外部獨立成一個.css檔


JavaScript

Javascript是用來增添網頁互動性的語言，例如點某個按鈕要執行什麼動作、或滑鼠滑到某個區塊要顯示什麼東西等等，都是靠javascirpt做出來的呦！

Javascript語法可以內嵌在.html裡，也可以放在.html外部獨立成一個.js檔

網頁設計的「設計」

很多人網頁設計的「設計」跟平面設計是同一件事，就圖文擺一擺、顏色調一調、介面看起來美美的就好。
但完全不是這麼回事！


互動

很容易理解的，網頁是不斷的在跟使用者互動的，游標、滾輪、鍵盤，這三件事就可以構成無數種互動形式，例如滑鼠的互動：互動事件有點擊(click)、懸浮(hover)、按住(active)等等，互動結果可能是點擊後跳出對話框、hover時文字變大icon變色、active時按鈕往下陷顏色變深等等。

而且近年來，互動設計變成網頁設計的重點，介面漂亮已經是最基本的要求了。例如小英的募款網站。如果是以傳統的平面設計思維來規劃網頁，那頗遜的。

那互動最終導向的價值是「使用者體驗」，使用者體驗就包含了這份網站的親合度、易用性、操作流程順不順、是否合乎直覺、有不有趣、跨螢幕體驗好不好、效能好嗎等等，這些事情是平面設計不會關注到的部分。
