網頁開發環境安裝

當我們在 Codepen.io 上練習完了之後，接下來要進入「真正的」網頁開發流程與環境了！

安裝文字編輯器

文字編輯器，顧名思義，就是編輯文字用的工具。

一般 Windows 會內建 Notepad，Mac 也有內建的 TextMate，都是能用的工具，但他們功能很陽春，拿來寫程式會比較沒效率。
於是市面上出現很多專為「程式設計」而打造的文字編輯器，推薦的有：Sublime Text、Notepad++、Atom 等等，選擇一個喜歡的安裝吧！


Let's Coding!

Before we start, 先了解網頁運作的原理：

每個網站背後其實都是由很多文字檔所組成的，你在瀏覽器打的網址，其實就是連到世界上某個角落的某台伺服器，然後讀取裡面的「某個文字檔」（例如index.html），再把那份檔案裡的程式碼傳到你的瀏覽器，在這裡進行解析（類似翻譯的感覺），然後就變成你在瀏覽器看到的東西囉！

所以，我們在寫網頁的時候，就是...
  1. 先在自己的電腦建立檔案（例如：index.html）。
  2. 利用文字編輯器開始編輯它，作出你想要的網頁。
  3. 想要測試的話，只要把你你正在編輯的檔案拖曳到瀏覽器裡，就能瀏覽你的 .html 檔案。
    也可以雙擊 .html 檔案，預設的瀏覽器就會自動打開。
  4. 完成編輯並且測試無誤後，就可以讓網站上線囉！
讓我們開始吧！

第一天的課程有提到，每個 HTML 檔都要有基本的架構，網頁瀏覽器才看得懂。為了免除學員們自己設定的困擾，我們已經製作好基本的 HTML 架構，下載之後就可以開始打造你的網頁了！

  • 下載基本架構

HTML 的基本架構就像一張白紙一樣，沒有任何元素或排版，如果要這樣土法煉鋼做出一個漂亮的網頁是很困難的，要是想再加入 RWD 的效果，往往會導致整個網頁壞掉。

多虧現在有很多網站提供現行的模版供使用者下載，如 HTML5 UP 或 Start Bootstrap。它們提供的模版包括框架和 RWD 的效果，只要將原本的內容換成自己想要的東西，再加上自己的創意，就能快速打造一個完整的網站。
